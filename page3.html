Installation LINUX

Installation OS Linux Mint

Boot Linux Mint
  1. Insérez votre clé USB dans l'ordinateur.
  2. Redémarrez l'ordinateur et taper la touche spéciale (F2 ou Del) pour entrer dans le BIOS.
  3. Modifier le BIOS pour démarrer sur la clé USB en mode EFI.
  4. A l'affichage un menu grub, appuyez [Enter] pour démarrer Linux Mint à partir de la clé USB.
Installation de Linux Mint sur l'ordinateur
  1. Double-cliquez sur l'icône [Installer Linux Mint].
  2. Sélectionnez votre langue.
  3. Se connecter à Internet.
  4. Cochez la case pour installer les codecs multimédias.
  5. Si Linux Mint est le seul système d'exploitation que vous souhaitez exécuter et que toutes les données peuvent être perdues sur le disque dur, choisissez [Effacer le disque et installer Linux Mint].
  6. Sélectionnez votre fuseau horaire.
  7. Sélectionnez votre disposition de clavier.
  8. Entrez vos informations d'utilisateur : - Votre name (il est uniquement utilisé sur l'écran de connexion et de veille). - Votre username est votre nom de connexion (n'utilisez que des caractères minuscules, sans ponctuation ni accentuation) - Votre hostname est le nom de votre ordinateur sur le réseau. - Pour protéger vos données personnelles contre les attaques locales, cochez [Chiffrer mon dossier personnel]. - Choisissez un mot de passe (il vous sera demandé à chaque démarrage ou mise à jour du système)
  9. Profitez du diaporama pendant que Linux Mint s'installe sur votre ordinateur.
  10. Lorsque l'installation est terminée, cliquez sur [Redémarrer maintenant].
Personnalisation de l'installation
  1. Installer les codecs multimédias.
  2. Installer/Supprimer les langages du système.
  3. Effectuer les mises à jour.
  4. Paramétrer le thème de l'ordinateur.
  5. Installer les applications supplémentaires.
  6. Placer les raccourcis sur le panneau.
  7. Créer une sauvegarde instantanée du système (Timeshift).
