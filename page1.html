Bien démarrer en HTML-CSS

Paramétrage VsCodium

Installer les plugins
  1. Live Server (Affichage du document dans le navigateur)
  2. Prettier Code formatter (Indentation auto des balises HTML)
  3. Material Icon Theme (place une icone devant les noms de fichier HTML-CSS-JS...)
Manage - setting
  • Autosave - onWindowChange (Sauvegarde auto au changt de fenètre)
  • Format on save - yes (Indentation auto à la sauvegarde)
  • WordWrap - on (Retour à la ligne auto)

Construire le fichier HTML

Attributs de texte, raccourcis VsCodium
  • <b>Texte en gras </b>
  • <em>Texte en italique </em>

  • ! Structure de base du doc HTML
  • Ctrl + / Transforme le texte en commentaire
  • lorem20 : Lorem, ipsum dolor sit amet consectetur adipisicing elit. Animi, rerum repudiandae beatae culpa quia error aliquam? Necessitatibus saepe laborum eveniet!
Liens utiles pour les emojis, icônes, images, videos, audio
  • 😉 Liste des émojis avec leur code UTF8
  • Liste de Liens utiles
  • Utiliser des icônes Fontawesome --> sur le site cdnjs vous trouverez le lien à copier dans le <head>
  • Images et Vidéos libres de droit : PixaBay --> <video src="./img/vagues.mp4" height="200" autoplay muted loop></video>
  • Ajouter de l'audio : Effets sonores --> <audio controls src="./audio/ocean-waves.mp3"></audio>

Construire le fichier CSS

Polices, Background, Couleurs
  • Utiliser ou Télécharger des Polices spéciales : FontGoogle
  • Choisir un fond d'écran ou une texture : Pixabay - Unsplash - - pexels - Websites for Creative Backgrounds --> {background: url(./img/text-mur-ciment-raye.jpg) center/cover}
  • Choisir une palette de couleur, en harmonie avec l'image de fond : Image picker
  • Choisir un dégradé de couleurs et générer son code CSS : CSS Gradient - uiGradients
  • Générateur d'effets sur le texte et les bordures
