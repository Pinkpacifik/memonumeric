Héberger son site web

Page Github

Préparation préalable

  1. Installer Git batch
  2. Créer un compte Github (le username sera utilisé dans l'adresse du site !)
  3. Espace alloué pour héberger le site sur Github (1 Go)

Créer un projet

  1. Créer un Nouveau Repository
  2. Ouvrir le terminal dans VsCodium et taper les command batch :
    git init ; git add . pour tout transférer.
    git commit -m "first commit"; git branch -M main ; git remote add origin [URL de la page .git] pour lier le dépot local et le dépot distant.
    git push -u origin main pour envoyer les fichiers locaux sur la branche distante
  3. Créer un site à partir de la branche main :
    Setting / Page [Select branch = main] / Save --> Récupérer l'adresse de la page.

Enregistrer les modifications du projet

Si des modifications sont apportées en local, lancer la commande :

git push dans le CHANGE bash, pour mettre à jour le dépot distant.

Lancez la synchronisation pour effectuer les mises à jour nécessaires.
